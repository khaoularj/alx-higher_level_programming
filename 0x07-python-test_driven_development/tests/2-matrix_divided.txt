def matrix_divided(matrix, div)
-----------------------

this function return a new matrix where the elements of the matrix are the result of division

Usage Example:
-------------

>>> matrix_divided = __import__('2-matrix_divided').matrix_divided

printing the matrix
>>> matrix = [1, 2, 3]
>>> print(matrix)
[1, 2, 3]

divising integers numbers
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> print(matrix_divided(matrix, 3))
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
